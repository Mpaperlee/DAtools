SequenceSearcher

SequenceSearcher is a JAVA application for search kmer sequence in a pre-format sequence database with a fasta file. This program was designed for Yong's lab. The first version was released in 2015.7.7 and authored by Qi Zhao. For beta version download, please contact me by email : Qi Zhao
